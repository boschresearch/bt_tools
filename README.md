# bt_tools

Tools to view behavior trees:

- [btlib](btlib/): Library to parse behavior trees.
- [bt_view](bt_view/): Tools to visualize behavior tree runs.
- [rqt_bt_live](rqt_bt_live/): Tool to view behavior trees in rqt at runtime.
